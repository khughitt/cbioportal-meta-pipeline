# Integrative analysis of complex cancer genomics and clinical profiles using the cBioPortal

- **Authors:** Gao J, et al.
- **Year:** 2013
- **Journal:** Science Signaling
- **PMID:** 23550210
- **BibTeX key:** Gao2013

## Key Contribution

A practical companion to Cerami 2012 that documents the full cBioPortal feature set as of early 2013, with step-by-step workflows for integrative queries, a REST-based web API (CGDS), and language bindings (R, MATLAB) for programmatic access. It also formalizes the Onco Query Language (OQL) and patient-level view, tying together mutation, CNA, mRNA, miRNA, methylation, and RPPA protein data.

## Methods

The portal exposes a CGDS (Cancer Genomic Data Server) REST-like interface returning tab-delimited text over HTTP, with commands including `getCancerStudies`, `getGeneticProfiles`, `getCaseLists`, `getProfileData`, `getMutationData`, `getProteinArrayInfo`, `getProteinArrayData`, and `getClinicalData`. A CRAN **CGDS-R** package returns results as R data frames, and a **CGDS Cancer Genomics Toolbox** on MATLAB Central mirrors each endpoint (tutorial via `showdemo cgdstutorial`). The web workflow is: pick cancer study, genomic profiles, case set, and gene list (optionally via MutSig/GISTIC lists or OQL filters such as `AMP`, `HOMDEL`, `MUT=V600E`, `EXP>x`, `PROT>x`). Results populate OncoPrint, Mutual Exclusivity (Fisher), Plots, Mutations (Mutation Mapper with Pfam domains, COSMIC, Mutation Assessor, Oncotator canonical isoforms), Protein Changes (t-test on RPPA), Survival (Kaplan-Meier + logrank), Network (Cytoscape Web), IGV, Download, and Bookmark tabs.

## Key Findings

The paper demonstrates capabilities through case studies: CDKN2A/CDK4/RB1 in glioblastoma show alterations in 71% of cases with near mutual exclusivity (CDKN2A-CDK4 and CDKN2A-RB1 significant, p<0.05); ERBB2 in colorectal cancer shows mRNA-protein correlation in amplified samples plus a V842I kinase-domain hotspot (4/10 mutations); PTEN loss in glioblastoma correlates with increased AKT phosphorylation (pT308, pS473) via the Protein Changes tab; BRCA1/BRCA2 mutations associate with significantly better overall and disease-free survival in ovarian cancer; and the network view of EGFR/ERBB2 in serous ovarian cancer reveals MYC amplified in 30% of TCGA samples and highlights drug targets (gefitinib, erlotinib, cetuximab, trastuzumab). A patient view integrates clinical attributes, recurrent alteration filters (MutSig/GISTIC, COSMIC, Cancer Gene Census), and links to drugs and cancer.gov trials.

## Relevance

Reference for programmatic access patterns; cite alongside Cerami 2012. Provides the concrete CGDS endpoint names, the CGDS-R CRAN package, and OQL syntax needed to script reproducible data pulls into the meta-analysis pipeline.

## Limitations

Explicitly out-of-scope at publication: cross-cancer queries for mRNA expression or protein abundance; mutations affecting only noncanonical isoforms (excluded from graphical summaries because calls are standardized to canonical RefSeq via Oncotator); and multi-gene correlation analyses beyond the queried gene set (users are pointed to Regulome Explorer and Oncomine). Genome-wide multi-track visualization is deferred to IGV and the UCSC Cancer Genome Browser. No rate limits are documented. The network view at the time required Adobe Flash (slated for removal mid-2013), and IGV required a Java Runtime. Data currency depends on the Broad GDAC Firehose pipeline and periodic TCGA releases (10 published studies plus ~20 TCGA studies in-pipeline as of Feb 2013).
