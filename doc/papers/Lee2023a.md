# Pan-cancer mutational signature analysis of 111,711 targeted sequenced tumors using SATS

- **Authors:** Donghyuk Lee, Min Hua, Difei Wang, Lei Song, Tongwu Zhang, Xing Hua, Kai Yu, Xiaohong R. Yang, Stephen J. Chanock, Jianxin Shi, Maria Teresa Landi, Bin Zhu
- **Year:** 2023
- **Journal:** medRxiv preprint (posted 2023-05-18; updated July 2024)
- **DOI/URL:** https://doi.org/10.1101/2023.05.18.23290188
- **BibTeX key:** Lee2023a
- **Source:** PDF

## Key Contribution

SATS (Signature Analyzer for Targeted Sequencing) is a new R package for mutational signature detection and burden estimation specifically designed for targeted gene panels, where sparse mutation data and heterogeneous panel sizes make WES/WGS-optimized methods (NMF, SigProfiler, deconstructSigs) unreliable. SATS extends canonical NMF to a Poisson Non-negative Matrix Factorization (pNMF) model that normalizes by a per-sample panel context matrix **L** (trinucleotide opportunities in megabase pairs), enabling analysis across mixed-panel cohorts and producing tumor-mutation-burden (TMB) signatures rather than raw mutation-count signatures. Applied to 111,711 AACR Project GENIE tumors across 23 cancer types, SATS generates the first pan-cancer mutational signature catalogue specifically calibrated for targeted sequencing, and demonstrates that per-sample signature burdens can be estimated even for a single tumor from this reference catalogue.

## Methods

**Model.** SATS uses a pNMF model where the SBS count $v_{pn}$ for mutation type $p$ in tumor $n$ follows Poisson($\ell_{pn} \sum_k w_{pk} h_{kn}$). The panel context matrix **L** (N × 96) records the number of callable trinucleotide contexts per panel, so the resulting signature profiles **W** are TMB signatures (mutations per Mb per trinucleotide), not raw mutation counts. When all samples share the same genomic region (WES/WGS), the pNMF reduces to canonical NMF.

**Two-stage pipeline.**
1. *Signature detection:* signeR (a pNMF-based de-novo extractor) is run on the cohort; the resulting de-novo profiles are mapped to COSMIC reference signatures using penalized non-negative least squares (pNNLS).
2. *Signature refitting (individual-level):* An Expectation–Maximization (EM) algorithm refits the cohort-detected COSMIC reference signatures to each patient's sparse mutation vector to estimate per-patient signature burdens (expected mutation count attributed to each signature).

**Data.** Primary cohort: AACR Project GENIE v13.0-public, 111,711 primary/metastatic tumors from 16 hospitals, 23 cancer type groups, 757 OncoTree subtypes; 982,095 SBS and 15,149 DBS mutations after quality filtering (read depth ≥ 100, alt allele count ≥ 5). Validation: GENIE v15.1 (13,425 additional lung/colorectal/breast tumors) and an independent Chinese cohort (2,143 lung + 1,206 colorectal + 306 breast). Benchmarking: pseudo-targeted data derived from TCGA WES and 560 breast WGS samples; in silico simulations up to 1M tumors across 21 panel sizes (> 1 Mb each); 72 kidney tumors with matched WGS + targeted sequencing.

**Comparators.** SigProfilerExtractor (de-novo), SigProfilerAssignment (refitting), Mix (clustering-based detection), deconstructSigs (refitting).

## Key Findings

1. **Spikiness and panel size govern detectability.** Detection probability is inversely correlated with the Shannon equitability index of a signature's TMB profile (r = −0.452): spiky signatures are easier to detect than flat ones. Within a cancer type, cancer type accounts for 53.26% of detection-probability variance (GLMM logit scale); larger panels (OR = 1.21 per Mb) and higher prevalence (OR = 1.12 per 1%) also improve detection.

2. **SATS outperforms competing methods on in silico benchmarks.** In simulated lung, breast, colorectal, and hematologic cancer data, SATS identified all pre-specified signatures for lung and breast (all replicates), 5/6 in colorectal, and all in hematologic cancer. SigProfilerExtractor missed SBS29 and SBS89 in lung and produced false-positives (SBS24 in lung, SBS7b in hematologic). Mix missed the majority of signatures in hematologic cancer. For burden estimation, SATS achieved median Pearson r = 0.70 (panels > 1 Mb) versus WES; other refitting methods performed worse, especially for low-prevalence signatures (SBS6 in colorectal r = 0.74 vs lower for SigProfilerAssignment/Mix).

3. **SATS recovers WGS-concordant burdens in matched samples.** In 72 kidney tumors with both WGS and targeted sequencing, SATS was the only method able to detect a significant correlation between WGS and targeted-sequencing burdens for the less common flat signature SBS1 (Wilcoxon p = 3.62 × 10⁻⁷); other methods assigned zero or cluster-level SBS1 burdens.

4. **Pan-cancer catalogue: signature landscape across 23 cancer types.** Key findings from the 111,711-tumor catalogue:
   - SBS1 (age/deamination) and flat signatures SBS3/5/40 are ubiquitous across all cancer types.
   - APOBEC signatures SBS2/13 predominate in bladder cancer; UV signatures SBS7a/b in skin/melanoma; tobacco signatures SBS4/29 and APOBEC SBS2/13 in lung; MMR-deficiency signatures SBS6/14/15 and POLE signatures SBS10a/b in endometrial cancer.
   - Treatment-induced signatures detected: SBS11 (temozolomide) in glioma, SBS32 (azathioprine) in pancreatic cancer, SBS87 (thiopurine) in lymphoid-derived hematologic cancer.
   - Seven DBS signatures identified; DBS1 (UV) in head/neck and skin; DBS2 (smoking) in bladder and lung.

5. **Validation across cohorts.** High replication in GENIE v15.1 for lung, breast, and colorectal signatures (SBS44 the main exception — difficult to separate from flat SBS5). Validated in Chinese cohort; SBS3 (HRD-like) detected in Chinese lung cancer where SBS44 was absent, suggesting cross-ethnic applicability.

6. **Clinical applications demonstrated.**
   - *Tissue-of-origin:* MSK-IMPACT samples clustered by cancer type using signature burdens; tumors of unknown primary clustered alongside lung, UV-signature tumors (head/neck, skin), and glioma — consistent with likely origins.
   - *Early-onset colorectal cancer:* In hypermutated colorectal cancers, early-onset was positively associated with SBS44 (deficient MMR; OR = 1.015, 95% CI = 1.002–1.027) and SBS10a (POLE; OR = 1.017, 95% CI = 1.000–1.034) but negatively with SBS5 (OR = 0.981, 95% CI = 0.963–0.999) — distinct mutational processes compared to late-onset.
   - *Immunotherapy response:* In 470 non-small cell lung cancer patients receiving immune checkpoint inhibitors, TMB attributed to SBS1 was associated with poor PFS (HR = 1.31, 95% CI = 1.10–1.56), while TMB from SBS4/29 (tobacco) was associated with favorable PFS (HR = 0.94, 95% CI = 0.90–0.98). Overall TMB showed only a nominal favorable association (HR = 0.98, 95% CI = 0.95–1.00).

## Relevance

**Hypothesis h08 (agnostic covariate–signature-exposure association; positive-control recovery of UV/smoking/APOBEC/MMR).** This paper is highly relevant on multiple levels:

- **Positive-control signal recovery (H08a).** The SATS catalogue demonstrates that the canonical exposure–signature links (UV↔SBS7 in skin, smoking↔SBS4/29 in lung, APOBEC↔SBS2/13 in bladder, MMR↔SBS6/14/15/44 in endometrial/colorectal, POLE↔SBS10 in endometrial) are robustly recoverable from targeted sequencing data at the cohort level. This is direct empirical evidence that h08's positive-control prong (H08a) is feasible even with sparse panel data, not just with WGS/WES.

- **TMB vs. total-mutation-count signatures.** SATS distinguishes "TMB signatures" (normalized by panel context, **W** profiles) from "TMC signatures" (raw mutation-count profiles from canonical NMF), and shows these differ for flat signatures like SBS5 and SBS10b. The cBioPortal pipeline should be aware that per-sample signature refitting on panel data requires panel-context normalization; naive NMF on raw counts conflates panel size with signature exposure — directly relevant to the design guard in h08 that "panels enter as cohort-pooled / refit, never per-sample."

- **Signature-specific TMB as a more informative immunotherapy biomarker than overall TMB.** The finding that SBS1-attributed TMB associates with *poor* PFS while SBS4/29-attributed TMB associates with *favorable* PFS, whereas overall TMB shows only a marginal signal, exemplifies h08b's core premise: agnostic covariate–signature association surfaces biologically coherent discoveries that overall-burden summaries obscure.

- **AACR GENIE as a substrate.** The pipeline already ingests AACR GENIE data (see `AGENTS.md`). The SATS catalogue (111,711 GENIE v13.0 tumors) could serve as an external validation reference for signature refitting in the cBioPortal cross-study meta-analysis, and the GENIE cohort's multi-panel diversity directly motivates the panel-context normalization problem SATS solves.

- **Cancer-type specificity of signature detection.** GLMM results show cancer type explains 53% of detection-probability variance — confirming that within-tissue stratification (the h08 design guard) is necessary to avoid tissue-collinearity confounding.

## Limitations

- The catalogue was built on GENIE v13.0 (2023 release); treatment-induced signatures (SBS32 in pancreatic cancer) may reflect population-specific treatment histories at contributing US/European centers and may not generalize to other registries.
- Flat signatures (SBS3, SBS5, SBS40) cannot be reliably distinguished from each other at current GENIE per-cancer-type sample sizes — the catalogue conflates them in practice.
- The study notes that hypermutated signature prevalence appears elevated relative to WGS/WES studies, possibly because targeted panels capture a higher fraction of the (sparse) hypermutated tumor's mutations in their narrow callable region.
- The immunotherapy PFS analysis is observational; confounding by cancer stage, treatment line, and institution is partially adjusted but not fully eliminated.
- The catalogue covers 23 cancer type groups (collapsed from 757 OncoTree subtypes); rare cancer subtypes remain underrepresented or unevaluated.
- SATS is an R package; no Python implementation is described, which may limit integration with the pipeline's Python-first stack.

## Follow-up

- **signeR (Rosales et al.)** — the de-novo pNMF extractor used inside SATS; reviewing signeR's model specification clarifies SATS's statistical assumptions.
- **SigProfilerExtractor / SigProfilerAssignment** — the main competing tools benchmarked here; understanding their weaknesses on sparse panel data motivates SATS adoption.
- **deconstructSigs** — another refitting comparator; performed poorly on kidney matched WGS/panel data.
- **AACR GENIE data guide (v13.0)** — documents mutation calling pipelines across the 16 contributing institutions, which Lee et al. note can impact signature detection (though less than burden estimation).
- Questions raised for h08:
  - Should the cBioPortal cross-study pipeline implement a panel-context-matrix normalization (analogous to SATS's **L**) before per-cohort signature refitting, to avoid conflating panel size with mutation burden?
  - Given that SBS1-attributed TMB is a *poor* immunotherapy-response predictor while SBS4/29-attributed TMB is *favorable*, should h08's covariate-association scan stratify by signature attribution rather than using overall TMB as a single covariate?
  - The SATS catalogue provides a GENIE-derived reference for signature refitting — could it be imported directly for the restricted-assignment step in `run_restricted_sigprofiler_assignment.py`?
