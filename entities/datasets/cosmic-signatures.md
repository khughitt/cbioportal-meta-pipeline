# COSMIC Mutational Signatures

## Summary

COSMIC Mutational Signatures [@Alexandrov2020] — the reference catalogue of SBS, DBS, and ID single/double/indel mutational signatures derived from PCAWG and other large cohorts; the standard basis for signature-decomposition analyses.

## Access and Scope

- Accessions: https://cancer.sanger.ac.uk/signatures/
- Source URL: https://cancer.sanger.ac.uk/signatures/
- Organism/population: Homo sapiens
- Modality: curated mutational-signature reference
- Sample size: ~80+ signatures (SBS/DBS/ID)
